# Project Rules: Zero to Hero ML Engineering Mastery

## Core Mandates

1.  **From Scratch**: All machine learning algorithms and logic must be implemented from `scratch`. usage of high-level frameworks (like PyTorch, TensorFlow, scikit-learn) is strictly forbidden unless explicitly authorized for a specific comparison module. We build to understand.
2.  **LaTeX Math**: All mathematical expressions, formulas, and derivations must be written in LaTeX. Documentation should be rigorous and textbook-quality.
3.  **Business Context**: Every lesson, example, and problem set must be applied to the context of **'Quantum Logistics Inc.'**, a futuristic logistics company.
    *   *Example*: Instead of predicting iris flower species, predict hyper-drive engine failure probabilities or optimize delivery drone swarm routes.

## Content Standards
- **Depth**: "World-class" quality. Deep dive into the 'why', not just the 'how'.
- **Code Style**: Python production-grade standards (typing, docstrings, modularity).
